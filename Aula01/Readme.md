# 📚 Aula 01 - Criação do Ambiente SQL Server

## 🎯 Objetivo

Nesta aula foi desenvolvida toda a estrutura inicial do banco de dados **DISCIPLINA02**, utilizando SQL Server.

O foco foi compreender como criar bancos de dados, tabelas, relacionamentos e restrições de integridade, formando a base para as próximas aulas da disciplina.

---

## 📖 Conteúdos estudados

- Criação de banco de dados (`CREATE DATABASE`)
- Seleção do banco (`USE`)
- Criação de tabelas (`CREATE TABLE`)
- Tipos de dados
- Chave Primária (`PRIMARY KEY`)
- Chave Estrangeira (`FOREIGN KEY`)
- Auto incremento (`IDENTITY`)
- Valores padrão (`DEFAULT`)
- Restrições (`CHECK`)
- Valores únicos (`UNIQUE`)
- Alteração de tabelas (`ALTER TABLE`)
- Coluna calculada
- Índices `CLUSTERED` e `NONCLUSTERED`
- `PRINT`
- `NEWID()`

---

- UFS

- CLIENTES

- VENDAS
- ITENS_VENDAS

## 🎓 Conhecimentos adquiridos

Ao concluir esta aula foi possível compreender:

- Como estruturar um banco de dados relacional.
- Como criar relacionamentos entre tabelas.
- Como aplicar regras de integridade utilizando constraints.
- Como utilizar diferentes tipos de índices.
- Como criar colunas calculadas.
- Como organizar um ambiente SQL Server para desenvolvimento.
